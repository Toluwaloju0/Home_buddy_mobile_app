Getting Started
clone this repo from github using the command
    `git clone `

run the following commands to install mongodb server and python3 on your local machine

    `sudo apt install --update -y python3 python3-pip python3-venv`
    `cd Home_buddy_mobile_app/backend/v1`
    `python3 -m venv .venv`   This is to ensure that all dependencies are used locally not from the global machine
    `source .venv/bin/activate`
    `pip3 install -r requirement.txt`
